Mais sobre o projeto

O site foi criado com HTML e CSS puro, sem frameworks ou bibliotecas externas, como forma de treinar conceitos de responsividade e design. Cada elemento foi planejado para manter uma estética limpa, moderna e interativa, mesmo sem uso de JavaScript.

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.
